# Инструкция по запуску фитнес-бота

## Шаг 1 — Создай бота в Telegram (2 минуты)

1. Открой Telegram, найди @BotFather
2. Напиши ему: /newbot
3. Придумай имя: например MyFitnessBot
4. Придумай username (должен заканчиваться на bot): например myfit_workout_bot
5. BotFather пришлёт токен — длинную строку вида:
   7123456789:AAHdqTcvCHhvQljS5m4tPVkDKq3v-xQz4K8
   СОХРАНИ ЕГО!

---

## Шаг 2 — Загрузи код на GitHub (3 минуты)

1. Зайди на github.com и зарегистрируйся (если нет аккаунта)
2. Нажми "New repository" → назови "fitness-bot" → Create
3. Нажми "uploading an existing file"
4. Перетащи все 3 файла: bot.py, requirements.txt, Procfile
5. Нажми "Commit changes"

---

## Шаг 3 — Запусти на Railway (5 минут, бесплатно)

1. Зайди на railway.app
2. Войди через GitHub
3. Нажми "New Project" → "Deploy from GitHub repo"
4. Выбери твой репозиторий "fitness-bot"
5. Railway сам обнаружит файлы и начнёт деплой

---

## Шаг 4 — Добавь токен бота

1. В Railway открой свой проект
2. Перейди в Variables (переменные)
3. Нажми "New Variable"
4. Введи:
   - Name: BOT_TOKEN
   - Value: вставь твой токен от BotFather
5. Нажми Add → Railway автоматически перезапустит бота

---

## Шаг 5 — Проверь!

Открой Telegram → найди своего бота по username → напиши /start

Готово! Бот работает 24/7.

---

## Как пользоваться ботом

/start — главное меню

### Кнопки:
- "Тренировка сегодня" — показывает упражнения на текущий день
- Кнопка ▶ у каждого упражнения — начать упражнение
- После нажатия → пиши результат в формате:

  подходы повторения вес

Примеры:
  4 15 18       (4 подхода × 15 раз × 18 кг)
  3 max 0       (3 подхода до отказа, без веса)
  1 10мин 0     (10 минут скакалки)

- "Мой прогресс" — статистика за 7 дней: подходы, средний вес, максимальный вес
- "Вся программа" — план на всю неделю

### Прогресс:
Бот автоматически сравнивает каждый результат с предыдущим и пишет:
🔥 Прогресс! — если вырос вес
✅ Держишь уровень! — если результат стабильный

## Расписание тренировок

Пн — Грудь (жим + разводка + отжимания)
Вт — Спина + бицепс (тяги + сгибания)
Ср — Отдых (растяжка + прогулка)
Чт — Плечи + трицепс (жим стоя + французский)
Пт — Кардио (скакалка интервалы)
Сб/Вс — Отдых

## Проблемы?

Бот не отвечает → проверь в Railway что процесс запущен (зелёный статус)
Ошибка токена → перепроверь переменную BOT_TOKEN в Railway Variables
